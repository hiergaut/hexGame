## hexGame
HEX game (university project with C and java)

Game under development ,not finished

Certainly finished in May 2017


## Usage

### Compilation

On linux, just use the Makefile,
at the root, use command :

```viml
make
```


### Execution

To play a game use

```viml
make run
```
